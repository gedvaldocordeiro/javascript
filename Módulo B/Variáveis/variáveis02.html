Treinando variáveis

Utilização do sinal de +

  • (number + number) adição
  • (string + string) concatenação

Comandos em JS

O comando prompt sempre retornará uma string mesmo que digite um número. Portanto não conseguimos fazer somas com os dados que recebemos desse comando.

Para convertemos uma string para number utilizamos o comando Number.parseInt('n') para números inteiros e para números com pontos ou virgulas utilizamos o Number.parseFloat('n').. Também existe o comando simplificado somente utilizando o Number('n'), porem nesse comando não definimos se ele é inteiro ou real. Veja o script abaixo:

            var n1 = Number.parseInt(window.prompt('Digite um número: '))
            var n2 = Number.parseFloat(window.prompt('Digite outro número: '))
            var n3 = Number(window.prompt('Digite outro número: '))
            var soma = n1 + n2 + n3
            window.alert('A soma dois números é, ' + soma)

Exitem dois comandos caso eu queira converter um number ou qualquer outro tipo primitivo em uma string e são eles o String('n') e o n.toString().Veja o script abaixo:

            window.alert('A soma dois números é, ' + soma.toString())
            ou
            window.alert('A soma dois números é, ' + String(soma))

Template String

O template string é uma forma mais simples de de escrevermos a nossa variável na tela. Com ela utilizamos a crase em vez das aspas simples e o símbolo do cifrão juntamente com chaves. Veja o exemplo: 

            `A soma de ${n1} e ${n2} é igual a ${soma}.`

Formatação de string

  • nome da variável.length: quantos caracteres tem a string
  • nome da variável.toUpperCase(): tudo para MAIÚSCULO
  • nome da variável.toLowerCase(): tudo para minúsculo

Formatação de Number

Com o .toFixed() conseguimos definir quantas casas após a virgula ou ponto queremos. Já com o .replace('' , '') conseguimos trocar uma coisa por outra. Veja: 

            var n1 = 15.6
            n1.toFixed(2).replace('.', ',')
            '15,60'
