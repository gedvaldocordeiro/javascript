Selecionando elementos no JavaScript

  1. Marca(tag)
    • Ex.: getElementsByTagName('p')[0]
  2. ID
    • Ex.: window.document.getElementById('msg')
  3. Nome
    • Ex.: window.document.getElementsByName('msg')[0]
  4. Classe
    • Ex.: window.document.getElementsByClassName('msg')[0]
  5. Seletor
    • Ex.: window.document.querySelector('div#msg')
